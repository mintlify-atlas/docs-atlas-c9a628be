## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** v14 or later — [nodejs.org](https://nodejs.org)
- **npm** v6 or later (bundled with Node.js)

Verify your versions:

```bash
node --version
npm --version
```

## Set up the game

<Steps>
  <Step title="Clone the repository">
    Download the project source to your local machine:

    ```bash
    git clone https://github.com/your-org/lucky-roll-dice.git
    cd lucky-roll-dice
    ```
  </Step>

  <Step title="Install dependencies">
    Install the required packages (Express for the server, reset-css for styles):

    ```bash
    npm install
    ```
  </Step>

  <Step title="Start the server">
    Launch the Express server using the start script defined in `package.json`:

    ```bash
    npm start
    ```

    You should see the following output in your terminal:

    ```
    Aplicación escuchando en el puerto 3000
    ```

    <Tip>
      The server defaults to port `3000`. To use a different port, set the `PORT` environment variable before starting: `PORT=8080 npm start`.
    </Tip>
  </Step>

  <Step title="Open the game in your browser">
    Navigate to the following URL:

    ```
    http://localhost:3000
    ```

    The Lucky Roll Dice game board will load in your browser.
  </Step>

  <Step title="Start a new game">
    Click the **New Game** button to initialize the board. This resets all scores to zero and gives the first turn to Player 1.
  </Step>
</Steps>

## Game controls

Once a game is started, three buttons control all gameplay:

<CardGroup cols={3}>
  <Card title="New Game" icon="rotate-ccw">
    Resets all scores to zero and begins a fresh game with Player 1 taking the first turn.
  </Card>
  <Card title="Roll Dice" icon="dice">
    Rolls the die and adds the result to the active player's current round score. Rolling a **1** forfeits the round score and passes the turn to the other player.
  </Card>
  <Card title="Hold" icon="hand">
    Banks the active player's current round score into their total score and passes the turn. The first player to reach **100 points** wins.
  </Card>
</CardGroup>

<Note>
  If a player rolls a **1**, their accumulated round score is lost and the turn passes immediately — without the option to hold.
</Note>
